This tool allows you to obtain high quality images from restricted sites for people on enterprise or school adminstered chromebooks.

Steps:
1. Search for your image in Google Images (it MUST be from Google Images)
2. Right click the image FROM THE GRID VIEW (Do not pull it up in the image preview pane)
3. Select "Copy Link Address"
4. Paste the address into the input area below
5. Copy your output URL
6. In a Google Document (or slideshow) insert an image BY URL. Use your output URL

Now you have it in a Google Document which is fine if you want it there, but if you'd like to have it as a file, you can right click the Google Document image and save it to keep. From keep, you can right click and save the image as a file
